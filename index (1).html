// FAQ

Frequently Asked Questions

Quick answers to the most common questions students ask.

Is it allowed to use portable software at school?
In most schools, running software from your personal user space is not prohibited — the restrictions are about installing software system-wide. Portable ZIP files do not install anything and do not require admin rights. That said, every school has its own IT policy, so it is worth checking yours if you are unsure.
Will this work on any Windows school computer?
Yes. As long as you can download files and open Command Prompt, the portable tools on this site will work. They have been tested on standard locked-down Windows 10 and Windows 11 school environments.
Can I put these tools on a USB drive?
Absolutely. That is one of the best ways to use portable software. Extract the ZIP to a USB drive, carry it with you, and use it on any school computer without downloading anything again.
Are the downloads safe and official?
Yes. This site only links to the official download pages of each tool — Oracle for Java, Python.org for Python, and so on. The files are not modified in any way. You are downloading the exact same file you would get from the official website.
What if something does not work?
Each tool page includes a detailed troubleshooting section. If you still have trouble, you can reach out through the Contact page and we will try to help.
