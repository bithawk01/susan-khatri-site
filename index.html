Three Steps to Run Java at School

Works on any Windows school or college computer — even locked-down ones.

01

Download the ZIP

Click any download button. The ZIP file saves to your Downloads folder — no permissions needed to download.

02

Extract It

Right-click → Extract All → choose your Documents folder or a USB drive. No admin password needed.

03

Set PATH & Run

Open CMD, set JAVA_HOME to your folder, and type java -version. Done. Full guide on the Java page.
