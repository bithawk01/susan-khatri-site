// Setup Guide

How to Set Up VS Code Portable

Follow these steps and VS Code will be running on your school computer in minutes.

01

Download the ZIP

Click the download button above. The file is around 100 MB so it may take a minute on slower connections. It saves to your Downloads folder — no permission needed.

02

Extract to a Folder

Right-click the ZIP → Extract All… → choose a location you own, such as Documents\VSCode or a USB drive. Do not extract it inside Program Files or anywhere that needs admin rights.

03

Enable Portable Mode

This is the key step. Inside the extracted folder, create a new folder named exactly data. This tells VS Code to run in portable mode and store all settings, extensions, and user data inside that folder instead of your Windows profile.

04

Launch VS Code

Double-click Code.exe inside the extracted folder. VS Code opens immediately — no installation wizard, no prompts, no admin password.

05

Install Extensions

Click the Extensions icon on the left sidebar (or press Ctrl+Shift+X). Search for and install any extension you need — Python, Java Extension Pack, Prettier, GitLens, and so on. They all install into your data folder without admin rights.

06

Open Your Project

Go to File → Open Folder… and select your project folder. VS Code remembers your recent projects and all your settings next time you open it.

Frequently Asked Questions

Common questions students ask about running VS Code on school computers.

Do I need to create the data folder every time?
No. You only create the data folder once. After that, VS Code remembers it is in portable mode and saves everything there automatically every time you open it.
Can I use VS Code with Java and Python portable too?
Yes, and it works great together. Install the Java Extension Pack and Python extension in VS Code, then point them to your portable Java or Python folder. All three tools can live on the same USB drive and work together without any installation.
Will my extensions still be there next time I open VS Code?
Yes, as long as you created the data folder before installing extensions. All extensions, settings, keybindings, and themes are stored in that folder and persist across sessions.
Can I move the VS Code folder to a different location?
Yes. You can move the entire VS Code folder anywhere — another location on the same computer, a USB drive, or a different computer entirely. Everything works because all data is self-contained inside the folder.
Is the portable version different from the regular VS Code?
No. It is the exact same application. The only difference is where data is stored. In the normal installed version, settings are stored in your Windows user profile. In portable mode, they are stored in the data folder you create. Features, performance, and extensions are identical.
